# Visualize.js
A basic audio visualizer plugin for jQuery.

Audio visualization has never been easier!
```
<div class="visualizeJs" src="test.mp3" style="width:500px;height:500px"></div>

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.1.0/jquery.min.js"></script>
<script src="visualize.js"></script>
```

All you need is to include the required libraries (jQuery) and our library (visualizer.js) and then create
a dive with a src attribute (in this case, test.mp3).  Then, you're good to go!  It is highly recommended that
you specify a width and height for the plugin.
